## **AAPL Stock Price Analysis (2023)**

This project analyzes Apple’s 2023 stock data and trains regression models to predict the closing price.

### **Main Steps**

* Loaded AAPL data using yfinance.
* Inspected data (head, shape, info, describe).
* Visualized trends and relationships (line plot, scatterplots, histogram, boxplot).
* Trained Linear Regression and Random Forest Regressor.
* Evaluated models using R², MAE, and MSE.
* Compared actual vs predicted closing prices.

### **Tools**

Python, pandas, seaborn, matplotlib, scikit-learn, yfinance
